## max(arg1, arg2, *args[, key])
* iterable の中で最大の要素、または2つ以上の引数の中で最大のものを返す。

* 位置引数が1つだけ与えられた場合、それは空ではない iterable でなくてはいけない。
その iterable の最大の要素が返される。

* 2つ以上のキーワード無しの位置引数が与えられた場合、その位置引数の中で最大のものが返される。

## iterableとは
for文で繰り返せるオブジェクト またはそのクラス
